# Notiq - Quick Notes Extension

Notiq is a convenient browser extension that allows you to create sticky notes on any web page. Save your thoughts, ideas, and reminders without leaving the current page.

## Features

- **Quick Sticky Notes** - Create notes instantly on any website
- **Drag & Drop** - Move notes anywhere on the page
- **Auto-Save** - Your notes are automatically saved
- **Multiple Colors** - Choose from yellow, blue, pink, or green notes
- **Minimal Design** - Clean and unobtrusive interface

## How to Use

- Click the Notiq icon in the toolbar to open the popup
- Toggle sticky notes on/off using the switch
- Select your preferred note color
- Click "New Note" to create a note
- Drag notes around the page by their header
- Click the minimize button to collapse notes
- Notes are automatically saved to browser storage

## Note Management

- View your 5 most recent notes in the popup
- Click on any note in the list to delete it
- All notes are stored locally in your browser

## Support

For issues or suggestions, please check the extension settings or reinstall the extension.
